void hwlib::ratio< T >::reduce ( )
inline

reduce the ratio

Calling this function reduces the ration to its canonical form by eliminiating the common divisors.

A ratio is always assumed to be in its canonical form. The ratio member functions call this function when needed. As a user, you must call this function after writing to the m and/or n members.
